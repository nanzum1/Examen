Título del proyecto

SIMULADOR PARA GESTIÓN DE INVENTARIO Y TRANSACCIONES DE BLACKMOUNSTER


Descripción del proyecto

La nueva Startup BlackMounster planea modernizar los sistemas tradicionales de arriendo
y venta de películas mediante un nuevo producto que promete mayor calidad de audio y
video. Para esto se genera sistema transaccional que gestione su inventario de
películas, permitiendo que estas sean arrendadas o compradas por los usuarios.


Entorno de desarrollo

Base de datos: PostgreSQL llamada db_blackmounster
Ambiente virtual: gestion_blackmounster utilizando D
Superusuario: nombre admin y contraseña
admin.

Lista de las tecnologías utilizadas

Django versión 4.1.1 
PostgreSQL, 
django-crispy-forms, 
crispy-bootstrap5, 
djangobootstrap-v5.
